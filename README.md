# Retail Sales Performance Dashboard

## Project Overview

This project analyzes retail sales data using SQL and Tableau to uncover sales trends, customer purchasing behavior, product category performance, and key business performance indicators (KPIs).

The dashboard is designed to help business leaders quickly monitor performance through interactive visualizations and executive-level KPIs.

## Business Questions

- Which product category generates the highest revenue?
- How do monthly sales change over time?
- Which gender contributes the most revenue?
- Which age groups generate the highest sales?
- Which product category sells the most units?
- What are the overall business KPIs?

---

## Key Insights

- Clothing generated the highest total revenue.
- Female customers contributed slightly more revenue than male customers.
- Monthly sales remained relatively consistent throughout the year.
- Customers spent an average of **$456** per transaction.
- Executive KPIs provide a quick overview of revenue, transactions, units sold, and average order value.
